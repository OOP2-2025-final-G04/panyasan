# アプリ名: パン屋さんの注文管理システム

概要:
パン屋さんの顧客管理や注文管理をする簡易アプリケーションです．

### 登録データ
    顧客データ
    User: id, name, age, point, birth_data, gender

    製品データ
    Product: id, name, price

    注文データ
    Order: id, user_id, product_id, order_data

    貯めたポイントの使い道，消費ポイントのデータ
    Pointhistory: id, user_id, used_point, used_at

### トップページのグラフ詳細
上段
- **左：棒グラフ**  
  性別・年齢別に分類した顧客情報を表示

- **右：円グラフ**  
  顧客によるポイント消費の割合を表示

中段
- **棒グラフ**  
  製品別の月間売上を可視化

下段
- **折れ線グラフ**  
  日別の売上推移（売上金額）を表示

## アピールポイント

年齢性別ごとにグラフ とパンの種類別売上グラフから誰にどのようなパンが人気なのかを予測しやすいようになっている．
日別の売り上げを表示することで，天気やイベントによる売り上げの変動が視覚的に分かりやすいようになっている．
顧客ごとの消費ポイントをランキング形式で可視化することで，利用状況が一目で分かるだけでなく，顧客同士の競争意識を刺激する．  
これにより，購買意欲の向上やリピート利用の促進が期待できる．
